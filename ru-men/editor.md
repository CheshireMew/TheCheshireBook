# 什么是比特币？

## 什么是比特币？

比特币是人类历史上第一次出现的、完全去中心化的数字货币。它由匿名开发者“中本聪”在 2009 年推出，不依赖银行、政府或任何中心机构，就能在全球范围内自由流通和交易。它不受单一国家或企业控制，其底层是区块链技术——一种去中心化、公开透明、不可篡改的账本系统。

### 一、比特币的核心特征

1. **总量恒定**：比特币的总发行量被固定在 2100 万枚，这种稀缺性让它被称为“数字黄金”。
2. **无需中介**：交易可以在全球用户之间直接发生，不需要银行清算。
3. **抗审查性强**：没有任何政府或机构可以冻结、没收你的比特币。
4. **公开透明**：所有交易都被记录在区块链上，任何人都可以查证。
5. **便于跨境流通**：不受汇率、国界、时间限制，几分钟即可完成全球转账。

### 二、比特币为何有价值

比特币的价值并不依赖法定货币的信用，而是来源于**共识与稀缺性**。它的每一枚代币都无法伪造，交易过程安全、公开、透明。\
随着越来越多的国家、企业、个人认可它，比特币逐渐具备了储值、支付、避险的功能，成为全球性的资产类别。

价值的三大支撑：

* 稀缺（总量固定）
* 共识（全球用户认同）
* 安全（区块链防篡改）
